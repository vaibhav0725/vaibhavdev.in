# Installing Next.js

Next.js is a popular React framework for building fast, user-friendly web applications. This guide will walk you through the steps to install Next.js and get your first project running.

## Prerequisites

Before you begin, make sure you have the following installed on your system:

- [Node.js](https://nodejs.org/) (version 16.8 or later recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

You can check if you have Node.js and npm installed by running:

```bash
node -v
npm -v
```

## Step 1: Create a New Next.js App

You can create a new Next.js application using the following command:

```bash
npx create-next-app@latest my-nextjs-app
```

Replace `my-nextjs-app` with your desired project name.

If you prefer using yarn:

```bash
yarn create next-app my-nextjs-app
```

## Step 2: Navigate to Your Project Directory

Change into your new project directory:

```bash
cd my-nextjs-app
```

## Step 3: Start the Development Server

To start the development server, run:

```bash
npm run dev
```

Or with yarn:

```bash
yarn dev
```

Your Next.js app should now be running at [http://localhost:3000](http://localhost:3000)!

## Next Steps

- Edit `pages/index.js` to start customizing your homepage.
- Read the [Next.js documentation](https://nextjs.org/docs) for more advanced features.

Happy coding!
